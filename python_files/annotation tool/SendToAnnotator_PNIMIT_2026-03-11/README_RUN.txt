CXR Differences Labeling Tool - End User Run Instructions

What you need
- Windows 10/11 (64-bit)
- No Python installation is required
- This folder must stay intact (do not move the EXE away from the pairs folder)

Folder contents
- CXR_Differences_Labeling_Tool.exe
- Pairs_PNIMIT_1_pairs\pair_A*_1_2 and pair_A*_2_3 (PNIMIT ready-to-load pairs)
- annotation tool guide.pdf
- Annotation Guidelines.docx
- Annotations\ (save JSON labels here)
- AnnotatedExports\ (optional exports)
- Collages\ (optional collages)
- OverlapReports\ (optional reports)

How to run
1. Double-click CXR_Differences_Labeling_Tool.exe
2. Click "Load Pair"
3. Browse to one pair folder inside Pairs_PNIMIT_1_pairs (example: Pairs_PNIMIT_1_pairs\pair_A1_1_2)
4. Label, then use "Save Labels" and save JSON files into Annotations\

Notes
- If Windows SmartScreen appears, click "More info" then "Run anyway".
- If the app fails with a missing runtime DLL, install "Microsoft Visual C++ Redistributable 2015-2022 (x64)" and run again.
